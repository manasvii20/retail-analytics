# CHOICES

## Why YOLOv8?

YOLOv8 provides fast real-time object detection with minimal setup and strong accuracy for person detection.

## Why ByteTrack?

ByteTrack offers reliable identity tracking and minimizes ID switching, which is important for retail analytics.

## Why FastAPI?

FastAPI provides:

* Automatic OpenAPI documentation
* Strong validation using Pydantic
* High performance
* Easy testing

## Why SQLite?

SQLite was chosen because:

* No setup required
* Lightweight
* Sufficient for hackathon-scale workloads

## Why Event-Driven Design?

Analytics are derived from events rather than raw video, making the system scalable and easier to integrate with downstream services.

## AI Usage

AI was used to assist in generating initial test scaffolding and implementation ideas.

All generated code was reviewed, modified, debugged, and validated manually.

Additional corrections were made for:

* Dependency injection
* Boundary conditions
* Idempotency validation
* Funnel computation
* Metrics correctness
